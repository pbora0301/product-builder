# Project Blueprint: Daily Word Lab

## 1. Overview

Daily Word Lab is a static Korean-language English vocabulary site. It provides five daily English words with pronunciation hints, Korean meanings, original example sentences, learning tips, and a short browser-only quiz.

## 2. Design and Style

The site follows a restrained editorial/tool layout:

*   **Layout:** Header navigation, first-screen learning value, daily word grid, learning method, quiz, editorial guide, and site policy information.
*   **Color Palette:** Neutral green-led palette with small gold and blue accents to avoid a one-note theme.
*   **Typography:** Large hero text, compact cards, and readable Korean body copy.
*   **AdSense Quality Direction:** Original content, clear navigation, no deceptive flow, visible operating information, balanced ad placement, and no thin utility-only page.

## 3. Features

*   **Daily Word Set:** A date-based word set renders automatically.
*   **Learning Cards:** Each card includes level, pronunciation, meaning, example, translation, and usage tip.
*   **Review Quiz:** Users answer five multiple-choice questions and receive instant scoring in the browser.
*   **Policy Sections:** Contact, advertising policy, and privacy guidance are visible on the page.

## 4. Project Structure

*   `index.html`: The main HTML file containing the structure of the page.
*   `style.css`: The stylesheet for the application.
*   `main.js`: The JavaScript file containing daily word selection and quiz logic.
*   `ads.txt`: AdSense publisher declaration.
*   `blueprint.md`: This file.
